Installing latest
===================

- Install apps dependencies (list in next section)
- Do following::

  $ git clone git@github.com:antigluk/ADCS.git; cd ADCS/src
  $ python2 setup.py install

- Run ADCS app::

  $ adcs

Dependencies
===============

apps
----------

- python>=2.7
- graphviz
- latex
- PyQt4

python
----------

(installed automatically with setuptools)

- pyYAML
- pydot
